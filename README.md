# JavaPub-Platform

从数据开发相关工作转到后台系统开发从一年多的时间，做后台系统开发，很想从零到一写一个后台管理系统开源出来。目前的想是先写一个基于 Spring Boot 的系统，后面再升级一版前后分离的（基于VUE），再升级一版分布式的（基于 Spring Cloud Alibaba）。
